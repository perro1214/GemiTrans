# GemiTrans - AI Page Translator

Gemini AIを使用してウェブページをリアルタイム翻訳するChrome拡張機能です。

## 機能

- ワンクリックでページ全体を翻訳
- 複数言語対応（日本語・英語・中国語・韓国語など10言語）
- 複数のGeminiモデル選択（Gemini 2.0 Flash / 2.5 Pro など）
- 自動翻訳モード（ページを開いた時に自動実行）
- バッチ翻訳による高速処理
- リトライ機能付き（レート制限・サーバーエラー対応）
- 翻訳のキャンセル・原文復元
- エラーログ表示

## インストール方法

1. このリポジトリをクローンまたはZIPでダウンロードします
   ```bash
   git clone https://github.com/perro1214/GemiTrans.git
   ```

2. Chromeで `chrome://extensions/` を開きます

3. 右上の「デベロッパーモード」をオンにします

4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、クローンしたフォルダを選択します

## 使い方

1. [Google AI Studio](https://aistudio.google.com/apikey) で Gemini API キーを取得します

2. 拡張機能のポップアップを開き、API キーを入力して「設定を保存」をクリックします

3. 翻訳したいページで「✨ ページを翻訳」ボタンをクリックします

4. ショートカットキー `Alt+T` でも翻訳を実行できます

## 設定項目

| 項目 | 説明 |
|------|------|
| API Key | Gemini API キー |
| 翻訳先の言語 | 翻訳後の言語を選択 |
| モデル | 使用するGeminiモデルを選択 |
| バッチ許容量 | 一度に送信する文字数の上限 |
| 自動翻訳 | ページ読み込み時に自動翻訳するか |

## 必要要件

- Google Chrome（最新版推奨）
- Gemini API キー（無料枠あり）
